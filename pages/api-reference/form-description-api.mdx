# FormDescription

`<FormDescription />` displays a description or helper text for a form field.

## Props

| Name      | Type      | Description                                 |
|-----------|-----------|---------------------------------------------|
| children  | ReactNode | The description text or content.            |
| component | string    | The HTML element to render (default: 'p').  |
| className | string    | Additional CSS classes.                     |

## Example

```tsx
<FormDescription>
  Please enter a valid email address.
</FormDescription>
```
