# FormMessage

`<FormMessage />` displays validation or error messages for a form field.

## Props

| Name      | Type      | Description                                 |
|-----------|-----------|---------------------------------------------|
| component | string    | The HTML element to render (default: 'p').  |
| className | string    | Additional CSS classes.                     |

## Example

```tsx
<FormMessage className="text-red-500" />
```
