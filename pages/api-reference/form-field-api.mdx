# FormField

`<FormField />` connects a field to React Hook Form context, handling value, validation, and error display.

## Props

| Name   | Type     | Description                                      |
|--------|----------|--------------------------------------------------|
| name   | string   | The field name in the form.                      |
| render | function | Render prop with field and formState parameters.  |

## Example

```tsx
<FormField
  name="email"
  render={({ field, formState }) => (
    <FormControl>
      <Input {...field} />
      <FormMessage />
    </FormControl>
  )}
/>
```
